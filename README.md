# Almadar Services Website

Marketing and documentation website for [Almadar Services](https://services.almadar.io) — Infrastructure for the agentic era.

Built with [Docusaurus](https://docusaurus.io/).

## Development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Deploy

Deploys automatically to Firebase Hosting on push to `main`.

```bash
npm run deploy
```
